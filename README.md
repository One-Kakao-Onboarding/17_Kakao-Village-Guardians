# PersonaTalk

> 관계의 맥락을 읽는 AI 기술로 현대인의 커뮤니케이션 피로도를 0(Zero)으로 만드는 것

카카오톡 스타일의 메신저에 **관계 기반 반응 엔진**을 추가한 혁신적인 커뮤니케이션 플랫폼입니다. 멀티 프로필을 기반으로 AI 페르소나를 결합해 말투, 격식, 감정, 시간, 취향을 자동 최적화하여 관계 유지 비용을 줄이고 표현 효율을 높입니다.

---

## 🎯 문제 정의

### 제1원칙 사고(First Principle Thinking)를 통한 문제 분해

**현상 (가정):**
- 사회적 관계(상사, 협력업체, 연인, 친구 등)에 따라 각기 다른 자아를 연기해야 한다
- 카톡 메시지를 보내기 전 문구와 말투를 고민하느라 시간이 오래 걸리고 스트레스를 받는다

**기본 요소 단위로 분해:**
- **요소 1:** 인간은 다중 자아(Persona)로 소통함 (상사, 친구, 연인, 가족, 거래처, 동호회 등)
- **요소 2:** 큰 의사소통 오류 비용 (텍스트 기반 소통의 맥락 부재)

**새로운 결론:**
카카오톡에 관계 기반 반응 엔진(감성적 기능)을 추가한다. 멀티 프로필을 기반으로 AI 페르소나를 결합해 말투, 격식, 감정, 시간, 취향을 자동 최적화하면 관계 유지 비용을 줄이고 표현 효율을 높일 수 있다.

---

### 5 Whys: 진짜 문제(Root Cause) 발굴

**현상:** 사람들이 카톡 메시지를 보낼 때 고민하고, 부담을 느끼고, 시간을 낭비한다.

- **Why 1:** 상대와의 관계에 맞는 적절한 '격식'과 '톤앤매너'를 맞추지 못하면 오해를 사거나 사회적 실수를 할까 봐 두렵기 때문이다.

- **Why 2:** 텍스트 중심의 소통은 비언어적 맥락(표정, 말투)이 부족하여, 조금만 말투가 어긋나도 '공격성'이나'무성의함'으로 오해받기 쉽기 때문이다.

- **Why 3:** 사용자는 사회적 위치에 따라 여러 개의 자아(Multi-Ego)를 가지고 있지만, 현재의 메신저 입력창은 모든 관계에 똑같은 빈 화면을 제공하기 때문이다.

- **Why 4:** 현재의 시스템은 '나의 프로필(정체성)'과 '메시지 입력(행동)'이 분리되어 있어, 사용자가 매번 머릿속으로 해당 관계의 규칙을 수동으로 인지하고 출력해야 하기 때문이다.

- **Why 5:** 결과적으로, 메신저가 사용자의 '사회적 맥락(Context)'을 이해하지 못한 채 단순한 텍스트 전달 도구로만 작동하고 있기 때문이다.

**최종 정의된 '진짜 문제':**

> **"메신저 인터페이스와 사용자의 사회적 페르소나 간의 동기화 부재"**
>
> 사용자가 관계별로 매번 수동으로 '자아를 전환(Persona Switching)'해야 하는 인지적 부하와 사회적 리스크가 소통의 효율성을 저해하고 있음.

**참고 자료:**
- 출처: '세대별 업무 카톡 스트레스 비율', 국내 20~50대 직장인 1000명, 오픈서베이 2021.05.05.
- MZ세대가 업무용 카카오톡 사용 시 스트레스를 받는 주요 이유

### 타겟 작업 분석

PersonaTalk가 해결하는 작업 유형:

- **고마찰(High Friction) 작업:**
  - 상황에 맞는 격식 차리기 (다/나/까 변환)
  - 감정 섞인 대화에서 실수하지 않기 (검열)
  - 늦은 시간 메시지 전송 여부 고민하기

- **고빈도(High Volume) 작업:**
  - 반복적인 안부 인사 (밥 먹었어?)
  - 단순 리액션 (이모티콘 선택)
  - 수많은 단톡방에서의 상황별 대답

- **고맥락(High Context) 작업:**
  - 상대방의 취향 (좋아하는 캐릭터) 반영
  - 이전 대화 톤 (정서적 지지형 vs 결과 중심형) 유지
  - 상대와의 친밀도에 따른 격식 지수 조정

**우리의 타겟 작업:**

> "멀티 프로필별 관계의 맥락을 학습하여, 상황에 맞는 말투·이모티콘·전송 매너를 자동 가이드하는 '커뮤니케이션 가드레일 및 페르소나 튜닝'"

---

## 💡 솔루션

PersonaTalk는 다음 핵심 요소로 문제를 해결합니다:

1. **멀티 프로필 시스템**: 공적 자아와 사적 자아를 분리하여 관리
2. **AI 기반 말투 변환**: 관계와 상황에 맞는 격식과 톤앤매너 자동 조정
3. **감정 가드 시스템**: 공격적이거나 부적절한 표현 사전 감지 및 순화
4. **스마트 리액션 봇**: 맥락을 이해한 이모지, 빠른 응답 자동 추천
5. **시간 기반 전송 제어**: 비매너적 시간대 전송 방지 및 예약 발송

## 👤 타겟 사용자 (페르소나)

### 니치(Niche) 시장 및 첫 고객(Beachhead) 선정

- **전체 시장**: 카카오톡을 사용하는 모든 스마트폰 사용자 (약 4,800만 명)
- **세분화 시장**: 공사 구분이 엄격하거나, 단톡방 개수가 20개 이상인 헤비 유저 (직장인, 커뮤니티 활동가)
- **니치 시장 (Beachhead)**:
  - "상사나 거래처와의 카톡 한마디에도 썼다 지우기를 반복하며 식은땀을 흘리는 사회초년생"
  - "공적 관계가 많아 멀티 프로필을 3개 이상 사용 중인 프로페셔널 직장인"

### AI PM용 페르소나 정의

**이름/직무:** 김도윤 (28세) / IT 기업 마케팅팀 1년 차 주니어 사원

**AI 숙련도(AI Literacy):**
- [활용 가능] ChatGPT를 업무 요약에 써본 적은 있으나, 대화 상황마다 적절한 프롬프트를 입력하기엔 번거로움을 느낌

**데이터 접근 권한:**
- [사원급] 팀 내 공지사항, 협업 도구 사용 권한은 있으나 결정권이 낮아 상사/거래처와의 커뮤니케이션에 조심스러움

**핵심 고통(Pain Point):**

> "상사에게 온 카톡에 '네'라고 할지 '네!'라고 할지 5분 동안 고민해요. 가끔 친구랑 카톡하다가 말투를 착각해서 거래처에 실수를 할까 봐 늘 긴장 상태고, 퇴근 후에도 단톡방 알림이 오면 예의 바르게 대답해야 한다는 압박감이 너무 커요."

---

### 유저 스토리(User Story)

**[유저 스토리 1: 사회적 실수 방지]**

- **As a** 상사나 거래처와의 카톡 한마디에도 식은땀을 흘리는 사회초년생은,
- **I want to** 밤늦은 시간 전송이나 공격적인 말투를 AI가 사전에 감지하고 제어해 주길 원한다,
- **So that** 비매너적인 행동으로 인한 관계의 실수를 방지하고 심리적 안정감을 얻기 위해.

**[유저 스토리 2: 페르소나별 말투 최적화]**

- **As a** 여러 개의 멀티 프로필을 운영하며 공사와 사를 엄격히 구분하는 직장인은,
- **I want to** 프로필에 설정된 페르소나에 맞춰 내 말투를 실시간으로 교정받고 싶다,
- **So that** 상황에 맞는 격식을 차리는 데 드는 인지적 에너지와 시간을 90% 이상 줄이기 위해.

## ✨ 주요 기능

### 프로필 및 페르소나 시스템

- **기본 말투 설정 (Global Default Tone)**: 앱 전반에 적용될 사용자의 기본 말투 모드 설정
- **관계 기반 멀티 프로필 (Relationship-based Multi-Profile)**: 대화 상대방별로 별도의 프로필 사진 및 말투 모드 지정
- **관계 태깅(Tagging)**: 상대방을 '친구', '상사', '동료', '선배', '가족' 등으로 정의하여 AI 변환 로직의 가중치로 활용
- **격식 지수 (Formality Level)**: 관계별 자동 격식 수준 설정 (0~100)
  - 상사: 95%, 선배: 70%, 동료: 50%, 친구: 10%, 가족: 15%

### 메시징 및 제어 (Messaging & Control)

#### AI 기능 제어
- **AI 기능 토글 (AI Assistance Toggle)**: 채팅방 내부에서 AI 개입 여부를 실시간으로 On/Off
  - Off: 일반 메신저처럼 raw text 입력
  - On: 설정된 페르소나로 자동 변환

#### 실시간 텍스트 변환
- **AI 말투 변환 엔진**: 사용자의 거친 입력(초안)을 설정된 관계 모드에 맞춰 즉시 윤문
- **변환 미리보기**: 원본과 변환된 텍스트를 비교하여 선택 가능
- **변경 사항 설명**: AI가 무엇을, 왜 변경했는지 상세히 표시

#### 감정 가드 시스템 (Emotion Guard)
- **공격성 감지**: 욕설, 비하, 공격적 표현 실시간 탐지
- **시각적 피드백**: 입력창에 빨간색 플래시 효과로 즉각 경고
- **자동 순화**: AI 말투 변환 API를 통해 부드러운 표현으로 자동 변환
- **원본 메시지 보존**: 순화된 메시지와 원본 메시지 비교 가능

#### 시간 기반 전송 제어
- **비매너 시간 감지**: 늦은 밤/이른 아침 전송 시 자동 경고
- **예약 발송**: 적절한 시간대로 자동 예약 발송 제안
- **스케줄 이유 설명**: 왜 예약 발송이 필요한지 상세한 설명 제공

### 스마트 리액션 봇 (Smart Reaction Bot)

#### 맥락 인식 엔진 (Context-Aware Engine)
- 상대방의 마지막 메시지 의도 분석 (질문, 요청, 거절, 공유 등)
- 감정 분석: 긍정적, 부정적, 놀라움, 축하, 응원 등

#### 3-Way 리액션 생성
1. **이모지 추천**: 대화 분위기에 맞는 이모티콘 세트 (최대 4개)
2. **빠른 응답(Quick Reply)**: 격식/비격식 모드에 맞춘 완성된 문장형 답변 추천 (최대 2개)

#### 이모티콘 시스템
- 카카오 스타일 이모티콘 지원
- 이모티콘 팩 관리
- 채팅 내 이모티콘 전송 및 표시

### 채팅방 관리

- **채팅방 목록**: 프로필별 필터링, 읽지 않은 메시지 카운트
- **실시간 폴링**: 3초마다 새 메시지 자동 확인
- **읽음 상태 추적**: 메시지별 읽음/안 읽음 표시 (isRead 필드)
- **채팅방 삭제**: 불필요한 채팅방 삭제 기능
- **메시지 리액션**: 메시지에 이모지 반응 추가


**PersonaTalk** - 관계의 맥락을 읽는 AI로, 당신의 커뮤니케이션을 자유롭게 만듭니다. 🚀
